## <a name="what"></a> 什么是冥想？
简单的说，冥想是一种技能，一种训练我们大脑的方式，让我们拥有更平静更清晰的思维，以及更放松的心态、身体及人生。

本质上，我们只是定期让自己脱离生活几分钟或者更久，暂停生活的脚步，训练大脑。让它更加感受当下，减轻纷扰。

## <a name="1"></a> 平静之心
修炼平静之心方法之一是进行专注冥想。

在训练大脑时，设定一个聚焦注意力的点是十分重要的，它可以是一个想象视觉化、一个问题、一个词语短句，甚至可以是外部的某个东西。最常见的专注点就是呼吸。

下面是具体实践方式。

### 准备阶段
1. 找到一个空间，保证接下来几分钟不被打扰。坐着或躺着都可以，具体姿势不强制要求，只要背部稍微挺直就好，让自己找到一个舒适的状态。
2. 双眼睁开，保持柔焦，不要盯着某个东西看，只是觉察自己身边的空间，做几次较大的深呼吸。用鼻子吸气，用嘴巴呼气。
3. 用鼻子吸气时，觉察到你的肺部充满气体，胸部在扩展。用嘴巴呼气的时候，感受身体的肌肉如何在放松。然后再做一次，用鼻子吸气，用嘴巴呼气的时候，轻轻地闭上眼睛。
4. 继续下一步之前花点时间暂停一下，享受这种停下沉静的感觉。这种停下来，不去思考自己的想法，不用思考任何事的感觉。
5. 不管坐着还是躺着，现在开始觉察身体压在座椅或床上的重量感。慢慢的有意识的察觉周围的一切，也许你开始察觉到周围的声音，让这些声音或思想自由来去，放下一切有意的抗拒，渐渐进入到这个空间里。利用这个机会观察自己的身体，觉察今天身体是否有沉重或轻盈的感觉，是否有躁动或沉静的感觉，慢慢感受身体的感觉，不用思考。

### 重点阶段

1. 然后开始察觉呼吸的起伏变化，保持这种状态，觉察你的身体有那些部位能感受到这种起伏变化的感觉。完全不用什么特殊的呼吸方式，如果感觉不到这种起伏，把手放在腹部。如果你愿意，可以随着吸气和呼气依次数数字，从 1 到 10 ，然后循环。如果发现自己的思绪开始游走，那么认可它，然后放下，再把注意力转回到呼吸上。这样持续一段时间。

### 收尾阶段
1. 最后继续保留这种感受，然后将意识拉回到身体上，回到身体与我们身下的座椅或床之间的碰触感和重量感上。把意识拉回到身边的声音中，准备好后，慢慢的睁开眼睛。
2. 在起身之前，花一点时间去察觉大脑的感受，察觉身体的感受，意识到这种感觉很重要，这样才很有可能再次回来这样练习。

将这种冥想中获取的感觉和心态，有意的带到你接下来的生活中。

## <a name="2"></a> 学会放下

2. 想象阳光会填满身体，填满的时候，会融化所有的不适和压力。

## <a name="3"></a> 感激当下

## <a name="4"></a> 应对压力

## <a name="6"></a> 应对痛苦
应对痛苦的方法之一是进行**身体扫描**。

身体扫描本质上可以帮我们让身体和心灵重聚。从头顶开始，向下扫描一直到脚趾感觉身体的感受，察觉那些部位舒适，那些部位不舒适。然后去适应它，不是抵触它，尽可能对这些感觉的保持好奇和兴趣。这些是为在大脑中创建更多的空间，拉开不适感与观察不适感之间的距离。

1. 最后继续保留这种感受，然后将意识拉回到身体上，回到身体与我们身下的座椅或床之间的碰触感和重量感上。把意识拉回到身边的声音中，准备好后，慢慢的睁开眼睛。
2. 在起身之前，花一点时间去察觉大脑的感受，察觉身体的感受。花一点时间看看，你和不适感之间的拉开距离是什么感觉。

随着不断的练习，甚至可以在身体各部位间暂停，然后开始去注意形状、体温和疼痛的种类。我们可以在大脑中创建一些空间，让疼痛和不适去做它们需要做的事情，而我们不必纠结其中。

## <a name="7"></a> 应对愤怒
应对愤怒的方法之一是进行**慈悲冥想**。

前面也用过，但具体的方式有点差别。

下面是具体实践方式。

### 准备阶段
1. 坐着或躺着都可以，具体姿势不强制要求，只要背部稍微挺直就好，让自己找到一个舒适的状态。
2. 双眼睁开，保持柔焦，不要盯着某个东西看，只是觉察自己身边的空间，做几次较大的深呼吸。用鼻子吸气，用嘴巴呼气。
3. 用鼻子吸气时，觉察到你的肺部充满气体，胸部在扩展。用嘴巴呼气的时候，感受身体的肌肉如何在放松。然后再做一次，用鼻子吸气，用嘴巴呼气的时候，轻轻地闭上眼睛。
4. 继续下一步之前花点时间暂停一下，享受这种停下沉静的感觉。这种停下来，不去思考自己的想法，不用思考任何事的感觉。
5. 不管坐着还是躺着，现在开始觉察身体压在座椅或床上的重量感。慢慢的有意识的察觉周围的一切，也许你开始察觉到周围的声音，让这些声音或思想自由来去，放下一切有意的抗拒，渐渐进入到这个空间里。利用这个机会观察自己的身体，觉察今天身体是否有沉重或轻盈的感觉，是否有躁动或沉静的感觉，慢慢感受身体的感觉，不用思考。

### 重点阶段
1. 想象一个某种程度上让你感觉到沮丧恼火的人，坐在你前面大概两米远的距离。依然能意识到呼吸的律动。
2. 再想象那个人遇到困难，或者遇到什么烦恼，或者身体健康出现了问题，然后把自己感觉到开心的一切事物与那个人分享。保持这种想法，希望这个人免收烦恼。感受着期间的感觉变化。

### 收尾阶段
1. 最后继续保留这种感受，然后将意识拉回到身体上，回到身体与我们身下的座椅或床之间的碰触感和重量感上。把意识拉回到身边的声音中，准备好后，慢慢的睁开眼睛。
2. 在起身之前，花一点时间去察觉大脑的感受，察觉身体的感觉。
