# 常用命令

## 引子
用一些命令老是重新搜索，还是自己记录一下更快。

## Windows

### 查看自己的 IP
```
ipconfig
```

### 查看局域网已使用的 IP

先查看自己的 ip 。

windos图标 + R ，输入 cmd ，打开一个命令窗口，运行以下指令，一定要是这样的命令窗口才行，打开 powershell 直接运行会报错。下面的是 `192.168.2` 换成上一步查出自己的 ip 前三部分。
```
for /L %i IN (1,1,254) DO ping -w 2 -n 1 192.168.2.%i
```
等一段时间后，再运行:
```
arp -a
```
